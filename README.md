# Werkr — AI-Powered Trade Services Platform

KI-gestütztes Matching zwischen Auftraggebern und verifizierten Fachbetrieben in Deutschland.

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env.local` file:

```
ANTHROPIC_API_KEY=your_api_key_here
```

## Deploy on Vercel

1. Push to GitHub
2. Import in Vercel
3. Add `ANTHROPIC_API_KEY` in Environment Variables
4. Deploy

## Built with

- Next.js 14
- React 18
- Anthropic Claude API
- Vercel
